# Tawantinsuyu — The Inca Empire

A single-page educational website covering the history of the Inca Empire from its origins in the Valley of Cuzco to the Spanish conquest and its modern legacy.

## Hosting on GitHub Pages

1. Create a new GitHub repository (public).
2. Upload all four files: `index.html`, `styles.css`, `script.js`, `README.md`.
3. Go to the repository's **Settings** → **Pages**.
4. Under "Source," select the `main` branch and `/ (root)` folder. Save.
5. Wait about a minute. Your site will be live at `https://<your-username>.github.io/<repo-name>/`.

## Files

- `index.html` — page structure and content
- `styles.css` — styling
- `script.js` — mobile nav and scroll animations

## Notes

Images are pulled from Unsplash and Wikimedia Commons via external URLs, so no image files need to be hosted. If you'd rather self-host the images, download them locally, put them in an `images/` folder, and update the `src` attributes in `index.html`.
